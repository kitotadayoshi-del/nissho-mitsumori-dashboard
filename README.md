# 見積使用状況ダッシュボード

このフォルダ（`dashboard/`）だけで動く、静的（HTML/JavaScript）ダッシュボードです。

## 入力データ

- `dashboard/data/見積使用状況.csv`（タブ区切り）
- `dashboard/data/ニッショー73支店_整形.csv`（支店マスタ）

## できること

- 期間（全期間 / 21日〜20日区切りの月）で絞り込み
- 支店別 件数ランキング
- 担当者別 件数ランキング（支店で絞り込み可）

## ローカルで確認

最も確実なのは、ローカルで簡易サーバーを立てて `dashboard/index.html` を開く方法です。
（`file://` で直接開くと、ブラウザの制約で自動読み込みが失敗することがあります。その場合でも、画面の「ファイル選択」から読み込みできます。）

このリポジトリ内に Node.js の簡易サーバーを同梱しています（追加インストール不要）：

```powershell
node .\dashboard\server.mjs
```

起動後、表示されたURL（例: `http://127.0.0.1:5173/`）をブラウザで開いてください。

## GitHub Pages に公開

`dashboard/` をそのまま Pages の公開対象（例: `/docs` など）に置くと動きます。
公開後は `dashboard/data/見積使用状況.csv` を更新してコミットすれば、次回アクセス時に反映されます。
